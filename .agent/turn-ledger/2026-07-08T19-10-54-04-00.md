## Main finding

`engine.command()` already returns a result, so Market authority should be added as a small source-owned layer instead of rewriting the runtime.

The fragile point is this chain:

```txt
exchange action row
  -> interface-composition.activate
  -> nested action.command
  -> ctx.engine.command(...)
  -> nested command result currently discarded
  -> no snapshot.lastResult
  -> no exchange projection
  -> no renderer readback
  -> no fixture replay
```
