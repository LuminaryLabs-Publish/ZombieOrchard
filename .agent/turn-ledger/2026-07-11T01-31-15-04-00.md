## Main finding

`interface-composition.activate` calls child commands through `ctx.engine.command()`. The child command notifies subscribers before the parent completes, the child result is discarded, and the outer command notifies again. A failed build can therefore produce a parent `{ accepted: true }` result while the actual service rejected the mutation.
