## Summary

Selected ZombieOrchard as the oldest eligible documented repository after comparing all ten accessible Publish repositories, excluding `TheCavalryOfRome`, and confirming no eligible repository was new, ledger-missing, or missing root `.agent` state.

The audit traces one runtime snapshot through the canvas and HTML renderers. `engine.tick()` publishes one snapshot to subscribers and then creates a second snapshot for the browser host. The host mutates the canvas first and the DOM second, without a shared frame identity, projection revision, commit barrier, rollback result, or first-visible-frame acknowledgement. A reentrant subscriber or a renderer fault can therefore make observer, canvas, HTML, diagnostics, and successor scheduling disagree about one logical frame.
